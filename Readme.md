# Real-Time Chat Application

## Technologies Used
- **HTML**
- **CSS**
- **JavaScript**
- **Node.js** with **WebSockets**

## Project Description
This is a real-time chat application where users can:
- Join or create chat rooms.
- Send and receive messages in real-time using WebSockets.
